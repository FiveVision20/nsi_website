Les Terminaux

Utilité

Un terminal, généralement appelé " ou "PowerShell" sur Windows, est une interface en ligne de commande qui permet aux utilisateurs d'interagir avec le système d'exploitation en entrant des commandes textuelles. Voici quelques exemples d'utilisation d'un terminal sur Windows :

Exemples

Navigation dans le système de fichiers :

  1. cd chemin_dossier: Changer le répertoire de travail.
  2. dir: Liste les fichiers et dossiers dans le répertoire actuel.

Manipulation de fichiers :

  1. copy fichier.txt dossier\: Copiez le fichier.txt dans le dossier.
  2. ren fichier.txt nouveau_nom.txt : Renommez le fichier.

Gestion des processus :

  1. tasklist: Afficher la liste des processus en cours.
  2. taskkill /F /IM nom_processus.exe : Termine un processus spécifique.

Réseau :

  1. ping adresse_ip : Vérifiez la connectivité réseau avec une adresse IP.
  2. ipconfig : Affiche les informations de configuration réseau.

Gestion des utilisateurs :

  1. whoami: Affiche le nom de l'utilisateur actuel.
  2. net user : Afficher la liste des utilisateurs.

Fenêtres de services :

  1. net start nom_service : Démarrez un service Windows.
  2. net stop nom_service : Arrêtez un service Windows.
  3. PowerShell : Exécution de scripts PowerShell pour des tâches plus avancées.

Édition de fichiers texte :

  1. notepad fichier.txt : Ouvre le fichier dans le bloc-notes.
  2. tapez fichier.txt : Afficher le contenu du fichier.
